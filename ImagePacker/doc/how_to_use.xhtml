使い方

基本操作

  1. 画像ファイルを入力する
  2. 出力設定を行う
  3. パッキングを開始する

①Input

　画像ファイルの入力全般

　Add
ファイル選択ダイアログからリストに入力
※ウィンドウへドラッグ＆ドロップすることで直接入力することも可能

　Remove
リスト内の選択されたファイルを消去

　Clear
リストに登録された内容をすべて消去

②Output
　Mode
Write only ... ファイル出力のみを行う
Display only ... 簡易ウィンドウで表示のみを行う
Both ... 両方の操作を行う

　File settings
Dir ... 出力フォルダの設定
File ... 出力ファイル名の設定
Format ... 出力形式の選択
Format option ... 出力形式に応じたパラメータの設定

　Packer option
Padding pixel ... 画像毎に枠をつける
Align 2^n boundary ... サイズを２のべき乗にする

③Start
　
　パッキングの開始を行う
